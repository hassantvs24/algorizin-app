### Project setup
Clone the project to any directory where you do development work

`git clone https://github.com/hassantvs24/algorizin-app.git`

###### Install necessary package 
`npm install`

###### Run
`npm start`

###### Dev run
`npm run dev`

###### **Must install mongoDB in your system
